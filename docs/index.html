Quick Start

1

Clone Repository

git clone https://github.com/joefusco/procedural-enclosure-toolkit.git
2

Select Template

Browse /templates or /examples for your use case.

3

Customize

Replace {{placeholders}} with case-specific information.

4

Validate

./scripts/validate.sh your-document.md
5

File

Convert to PDF and file via NYSCEF or CM/ECF. Serve on all parties.
